# Как обновить существующий репозиторий BoomBoom

1. Распакуй архив `BoomArena_GitHub_Полный_проект_0.3.zip`.
2. Открой папку `BoomArena_GitHub_0.3`.
3. Скопируй **всё содержимое**, включая скрытую папку `.github`, в локальную папку репозитория с заменой файлов.
4. Запусти `ОБНОВИТЬ_GITHUB_0.3.bat`.
5. На GitHub открой **Actions → Build Android APK**.
6. После зелёной сборки скачай artifact `BoomArena-Android-APK`.
7. Внутри artifact будут APK и файл контрольной суммы SHA-256.
