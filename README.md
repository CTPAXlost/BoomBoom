# Boom Arena 0.5

Открытый прототип мобильного командного FPS на Godot 4.7.1. Проект рассчитан на Android и собирается через GitHub Actions.

## Что есть в версии 0.5

- Локальный бой 4 × 4: игрок, 3 союзных бота и 4 противника.
- Компактная карта **«Старая ферма»** с двумя фермерскими базами, травой, сеном, оградами, деревьями и облаками.
- Четыре слота основного оружия.
- Стартовый автомат, дробовик, резервный пистолет и нож.
- Ручная стрельба и отключаемая автострельба.
- Отдельная 3D-анимация удара ножом.
- Попадания в голову с повышенным уроном.
- Монеты за убийства и завершение боя.
- Магазин оружия, брони и улучшений.
- Броня за 500 монет: 100 брони поверх 100 HP.
- Прокачка автомата с 1 до 5 уровня.
- Отдельные настройки чувствительности и FPS: 30 / 60 / 120.
- Сохранение монет, покупок, улучшений и настроек.
- Автоматический поворот между левой и правой альбомной ориентацией.

## Управление на Android

- Левая часть экрана — плавающий джойстик движения.
- Правая часть экрана — поворот камеры.
- **ОГОНЬ** — удержание для стрельбы.
- **НОЖ** — ближняя атака с анимацией.
- **R** — перезарядка.
- **AUTO** — включение или отключение автострельбы.
- Кнопки 1–4 — выбор оружейного слота.

## Сборка APK на GitHub

1. Скопируйте всё содержимое проекта в репозиторий, включая скрытую папку `.github`.
2. Отправьте изменения в ветку `main`.
3. Откройте вкладку **Actions**.
4. Дождитесь завершения **Build Android APK**.
5. Скачайте артефакт `BoomArena-Android-APK`.

Workflow проверяет структуру проекта, собирает ARM64 APK, проверяет подпись и прикладывает SHA-256.

## Важное ограничение прототипа

Это ранняя локальная версия без сетевого сервера, сложной навигации ботов, готовых художественных 3D-моделей и окончательного баланса. Все ресурсы прототипа оригинальные и не копируют материалы Guns of Boom.
