# Как обновить репозиторий до Boom Arena 0.4

1. Распакуй архив проекта.
2. Скопируй **всё содержимое** папки `BoomArena_GitHub_0.4` в локальную папку своего репозитория с заменой.
3. Проверь, что перенеслись скрытая папка `.github` и папка `signing`.
4. Запусти `ОБНОВИТЬ_GITHUB_0.4.bat`.
5. Открой GitHub Actions и дождись завершения `Build Android APK`.
6. Скачай artifact `BoomArena-Android-APK`.

Перед первой установкой 0.4 удали старую Boom Arena из-за смены временной подписи на постоянную. После установки 0.4 следующие APK можно будет обновлять поверх неё.
