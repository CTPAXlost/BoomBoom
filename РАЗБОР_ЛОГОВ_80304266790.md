## Исправление 0.3

Workflow больше не полагается на жёстко заданный путь. Он:

1. выводит список созданных файлов;
2. находит `BoomArena-debug.apk` внутри `build/` на любой вложенности;
3. при необходимости находит первый APK;
4. проверяет, что файл непустой;
5. проверяет подпись через `apksigner`;
6. читает пакет через `aapt`;
7. копирует APK в `artifacts/`;
8. создаёт SHA-256;
9. загружает APK и контрольную сумму в GitHub Artifact.
