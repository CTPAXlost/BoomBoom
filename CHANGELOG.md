# Изменения

## 0.2.0 — исправление GitHub-сборки

- Устранён конфликт `VirtualJoystick` со встроенным классом Godot 4.7.
- В базовый класс бойца добавлены методы `on_health_changed()` и `on_respawned()`.
- Удалены Min SDK/Target SDK из не-Gradle Android preset.
- Включена ETC2/ASTC-компрессия для Android.
- Splash и Android-иконки переведены в PNG.
- Проверка репозитория теперь ловит эти ошибки до экспорта.
- После сборки APK проверяется через `apksigner` и `aapt`.
