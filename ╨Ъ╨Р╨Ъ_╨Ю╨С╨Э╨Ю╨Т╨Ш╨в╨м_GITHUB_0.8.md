# Обновление репозитория до Boom Arena 0.8

1. Распакуйте архив полного проекта.
2. Скопируйте **всё содержимое** папки `BoomArena_GitHub_0.8` в локальную папку вашего репозитория с заменой.
3. Включите отображение скрытых файлов и убедитесь, что скопировалась папка `.github`.
4. Убедитесь, что на месте папки `assets/audio`, `scripts`, `signing`, файлы `project.godot` и `export_presets.cfg`.
5. Запустите `ОБНОВИТЬ_GITHUB_0.8.bat`.
6. На GitHub откройте **Actions → Build Android APK**.
7. После зелёной сборки скачайте артефакт `BoomArena-Android-APK`.

Сохранение 0.7 мигрирует автоматически. Старое оружие и покупки сохраняются, новые автоматы, каска, карта и уровни добавляются в существующий профиль.
