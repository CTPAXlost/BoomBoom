# Разбор logs_80380196175

Сборка остановилась на шаге `Run real battle scene smoke test` до экспорта APK.

Godot сообщил:

- `Preload file "res://assets/audio/grenade.wav" has no resource loaders`;
- невозможность загрузить `player.gd` и `bot.gd`;
- вслед за этим `arena.gd` не смог создать боевую сцену;
- workflow дождался таймаута и завершился кодом 124.

Причина: smoke test запускал проект из чистого Git-репозитория до первого импорта WAV-файлов. `gdlint` проверяет синтаксис, но не выполняет импорт ресурсов Godot.

Исправление 0.8.2:

1. отдельный шаг `Import Godot resources` с `--import`;
2. только после него запускается боевая сцена;
3. необработанные WAV больше не используются через статический `preload()` в боевых скриптах;
4. при невалидном скрипте `main.gd` завершает тест сразу;
5. в артефакт диагностики попадают `godot-import.log` и `smoke-test.log`.
