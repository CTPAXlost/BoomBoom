# История изменений

## 0.5.0 — Старая ферма, броня и развитие оружия

- Добавлена отдельная кнопка **«Настройки»** в главном меню.
- Чувствительность сенсора перенесена в настройки.
- Добавлен выбор ограничения FPS: **30 / 60 / 120**.
- Значения чувствительности, автострельбы и FPS сохраняются.
- Полностью заменена тестовая арена на компактную карту **«Старая ферма»**:
  - две фермы по противоположным сторонам;
  - зелёная трава;
  - деревянные ограды;
  - сено, ящики и деревья;
  - движущиеся облака над картой.
- Добавлена броня стоимостью **500 монет**.
- Броня даёт **100 единиц** и принимает урон раньше HP.
- После возрождения купленная броня восстанавливается до 100.
- Стартовый автомат ослаблен на первом уровне.
- Добавлены показатели мощности и урона автомата для уровней 1–5.
- Стоимость улучшений автомата:
  - 1 → 2: 300 монет;
  - 2 → 3: 500 монет;
  - 3 → 4: 900 монет;
  - 4 → 5: 1500 монет.
- Добавлен повышенный урон при попадании в голову.
- Добавлено сообщение о попадании в голову.
- Добавлена видимая 3D-анимация удара ножом.
- Боты используют базовый уровень оружия и не получают улучшения игрока.
- Версия Android-пакета увеличена до 0.5.0 / code 5.
