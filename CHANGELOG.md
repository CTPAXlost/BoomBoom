# История изменений

## 0.8.2 — исправление GitHub-сборки и первого импорта

- Исправлен smoke test: Godot теперь сначала импортирует WAV и остальные ресурсы через `--import`, а уже затем запускает боевую сцену.
- Убраны статические `preload()` необработанных WAV-файлов, которые могли останавливать разбор боевых скриптов до первого импорта.
- `player.gd` и `bot.gd` напрямую наследуют `combatant.gd`, не полагаясь на ещё не созданный кэш глобальных классов.
- Проверки бойцов переведены на группу `combatants`, чтобы загрузка не зависела от порядка регистрации `class_name`.
- Перед созданием арены проверяется `Script.can_instantiate()`. При ошибке smoke test завершается сразу, а не висит 45 секунд.
- GitHub сохраняет два диагностических файла: `godot-import.log` и `smoke-test.log`.
- Валидатор запрещает возвращать небезопасные `preload()` аудиофайлов и проверяет наличие стадии импорта.
- Android: версия 0.8.2, versionCode 10.
