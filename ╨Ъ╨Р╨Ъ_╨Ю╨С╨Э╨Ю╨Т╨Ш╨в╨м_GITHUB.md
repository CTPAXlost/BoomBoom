# Как обновить существующий репозиторий BoomBoom

1. Распакуй архив.
2. Открой папку `BoomArena_GitHub_0.2`.
3. Скопируй **всё содержимое**, включая скрытую папку `.github`, в локальную папку репозитория с заменой файлов.
4. Запусти `ОБНОВИТЬ_GITHUB_0.2.bat`.
5. На GitHub открой **Actions → Build Android APK**.
6. После зелёной сборки скачай артефакт `BoomArena-Android-APK`.

Можно вместо копирования сразу распаковать архив в пустую папку и запустить BAT: он попросит HTTPS-ссылку репозитория.
