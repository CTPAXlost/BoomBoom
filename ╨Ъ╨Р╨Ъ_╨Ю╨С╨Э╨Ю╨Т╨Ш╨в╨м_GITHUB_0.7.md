# Обновление репозитория до Boom Arena 0.7

1. Распакуйте архив версии 0.7.
2. Скопируйте **всё содержимое** папки `BoomArena_GitHub_0.7` в локальную папку репозитория с заменой.
3. В Проводнике Windows включите `Вид → Показать → Скрытые элементы` и убедитесь, что перенеслась папка `.github`.
4. Не удаляйте папку `signing`: в ней находится постоянный тестовый ключ подписи.
5. Запустите `ОБНОВИТЬ_GITHUB_0.7.bat`.
6. Откройте на GitHub вкладку `Actions` и дождитесь зелёной сборки `Build Android APK`.
7. Скачайте артефакт `BoomArena-Android-APK` и установите APK.

Сохранение 0.6 должно автоматически обновиться. Новому профилю выдаются 30 аптечек и 30 гранат; существующий профиль без этих полей также получит стартовое количество при первом запуске 0.7.
