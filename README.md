# Boom Arena 0.8

Исходный проект локального мобильного командного FPS на Godot. В бою участвуют один игрок, три союзных бота и четыре противника. Проект предназначен для Android и собирается в APK через GitHub Actions.

## Режимы и карты

### Старая ферма — командный бой

Команда, первой набравшая 25 устранений, побеждает. Карта увеличена по сравнению с предыдущей версией: центральное направление разделено укрытиями, слева и справа появились отдельные маршруты, сараи, тюки сена и небольшие нычки.

### Салун — контроль зоны

- Цель — первой набрать 1000 командных очков.
- Центральная точка находится внутри салуна.
- Для захвата нужно непрерывно удерживать её 5 секунд.
- Захваченная точка приносит 5 очков в секунду.
- Когда в зоне находятся обе команды, захват и начисление очков приостанавливаются.
- После смерти боец возвращается на респаун своей команды.
- На каждом респауне предусмотрены два выхода.
- В салун можно войти с обеих сторон. Внутри находятся барная стойка, укрытия, второй этаж, балкон и подъёмная рампа.

## Очки режима «Салун»

- обычное устранение — 10 очков;
- первое устранение матча — 20 очков;
- устранение ножом — 15 очков;
- устранение в голову — ещё 25 очков;
- Double Kill — ещё 10;
- Triple Kill — ещё 20;
- Quad Kill — ещё 30;
- Unstoppable — ещё 50;
- каждое следующее устранение после Unstoppable — ещё 20.

Бонусы суммируются. Серия считается в пределах одной жизни и сбрасывается после смерти.

## Оружие

В четыре слота можно поставить любые купленные виды основного оружия.

### Автоматы

| Оружие | Открытие | Особенность |
|---|---:|---|
| AR-4 «Новобранец» | уровень 1 | стартовый сбалансированный автомат |
| VX-7 «Вихрь» | уровень 2 | высокая скорострельность, магазин 36 |
| BR-9 «Бастион» | уровень 3 | высокий урон, магазин 28, более сильная отдача |
| PX-12 «Феникс» | уровень 4 | дальность 28, магазин 40, точная стрельба |

Также доступны дробовик SG-12 «Гром», пулемёт MG-60 «Бизон», снайперская винтовка SR-70 «Егерь» и резервный пистолет.

### Перезарядка

| Тип оружия | Время |
|---|---:|
| Автомат | 2 секунды |
| Снайперская винтовка | 3 секунды |
| Дробовик | 4 секунды |
| Пулемёт | 5 секунд |

Боты расходуют патроны в магазине и ждут полную перезарядку наравне с игроком.

## Прогресс персонажа

В проекте пять уровней:

| Уровень | Требуемый общий опыт |
|---:|---:|
| 1 | 0 XP |
| 2 | 250 XP |
| 3 | 700 XP |
| 4 | 1400 XP |
| 5 | 2400 XP |

На втором уровне здоровье увеличивается со 100 до 200 HP. Новые уровни открывают автоматы, каску и дополнительные ступени брони.

## Снаряжение

- Броня развивается от 0 до 500 единиц.
- Тактическая каска открывается на уровне 3, стоит 2200 монет и уменьшает урон в голову на 50%.
- Аптечка лечит 10 HP. Доступно не больше 10 применений за одну жизнь.
- Граната наносит 100 урона. Доступно не больше двух бросков за одну жизнь.
- Потраченные аптечки и гранаты вычитаются из постоянного запаса.

## Выбор оружия перед боем

Во время отсчёта `5 → 4 → 3 → 2 → 1` игрок не может двигаться, стрелять или использовать предметы, но может нажать одну из кнопок оружейных слотов и выбрать стартовое оружие.

## Звук

Проект содержит оригинальные синтезированные WAV-звуки:

- два варианта шагов;
- разные выстрелы четырёх автоматов;
- отдельные звуки дробовика, пулемёта, винтовки и пистолета;
- перезарядку, нож и взрыв гранаты.

## Управление Android

- левая область — движение;
- правая свободная область — камера;
- **ОГОНЬ** — ручная стрельба;
- **ПРИЦЕЛ** — вход и выход из прицела;
- **НОЖ** — удар ножом;
- **АПТЕЧКА** и **ГРАНАТА** — расходники;
- **R** — перезарядка;
- **AUTO** — автострельба;
- кнопки 1–4 — выбор оружия.

Управление для проверки на ПК: WASD, мышь, левая кнопка — огонь, правая — прицел, R — перезарядка, Q — нож, H — аптечка, G — граната, F — автострельба, 1–4 — оружие.

## Сборка APK через GitHub

Workflow `.github/workflows/android.yml` выполняет проверку структуры, разбирает GDScript, экспортирует ARM64 APK, проверяет подпись и загружает артефакт `BoomArena-Android-APK`.

Постоянный тестовый ключ находится в `signing/boomarena-debug.keystore`. Не публикуйте этот ключ как боевой ключ приложения.

## Проверка

Перед упаковкой проекта запускаются:

```bash
python scripts/validate_project.py
gdlint autoload/save_data.gd scripts/**/*.gd
```

Физическое ощущение управления, расположение элементов интерфейса и поведение ботов окончательно проверяются на Android после сборки APK.
