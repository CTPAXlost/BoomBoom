# Как обновить репозиторий до Boom Arena 0.5

1. Распакуйте архив версии 0.5.
2. Скопируйте **всё содержимое** папки `BoomArena_GitHub_0.5` в локальную папку своего репозитория с заменой.
3. Убедитесь, что скопировалась скрытая папка `.github`.
4. Запустите `ОБНОВИТЬ_GITHUB_0.5.bat`.
5. Откройте GitHub → Actions → Build Android APK.
6. После зелёной сборки скачайте `BoomArena-Android-APK`.

Идентификатор приложения и постоянный тестовый ключ не изменились, поэтому версия 0.5 должна устанавливаться поверх 0.4 обычной кнопкой «Обновить».
