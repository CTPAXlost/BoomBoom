# Разбор GitHub Actions от 18.07.2026

Сборка остановилась на этапе `Export Android APK`. Найдены четыре блокирующие причины:

1. `VirtualJoystick` совпал с именем встроенного класса Godot 4.7.1. Класс переименован в `BoomTouchJoystick`.
2. Базовый `Combatant` вызывал методы, которых компилятор не видел в базовом классе. Добавлены пустые виртуальные хуки `on_health_changed()` и `on_respawned()`, которые переопределяются игроком и ботами.
3. В Android preset были указаны Min SDK и Target SDK при выключенном Gradle Build. Эти параметры удалены.
4. Для Android не была включена ETC2/ASTC-компрессия. Добавлен параметр `textures/vram_compression/import_etc2_astc=true`.

Дополнительно SVG убран из boot splash, потому что Godot 4.7 принимает для заставки PNG. Workflow теперь проверяет итоговый APK через `apksigner` и читает его манифест через `aapt`.
