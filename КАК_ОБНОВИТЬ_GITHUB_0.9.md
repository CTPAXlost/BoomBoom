# Как обновить Boom Arena до 0.9

1. Распакуйте архив `BoomArena_GitHub_Полный_проект_0.9.zip`.
2. Скопируйте всё содержимое папки `BoomArena_GitHub_0.9` в локальную папку вашего репозитория с заменой.
3. В Windows включите показ скрытых элементов и убедитесь, что скопирована папка `.github`.
4. Запустите `ОБНОВИТЬ_GITHUB_0.9.bat`.
5. Откройте GitHub → Actions → `Build Android APK`.
6. Дождитесь прохождения двух тестов:
   - `Run real battle scene smoke test`;
   - `Run training ground smoke test`.
7. Скачайте артефакт `BoomArena-Android-APK`.

Идентификатор приложения и тестовый ключ подписи не менялись, поэтому версия 0.9 должна устанавливаться поверх 0.8.2 как обычное обновление.
