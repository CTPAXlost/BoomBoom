# Обновление репозитория до Boom Arena 0.6

1. Распакуйте архив проекта.
2. Скопируйте всё содержимое папки `BoomArena_GitHub_0.6` в локальную папку вашего репозитория с заменой файлов.
3. Убедитесь, что перенеслась скрытая папка `.github`.
4. Запустите `ОБНОВИТЬ_GITHUB_0.6.bat`.
5. Откройте репозиторий на GitHub и перейдите в `Actions → Build Android APK`.
6. После зелёной сборки скачайте артефакт `BoomArena-Android-APK`.

Сохранение версии 0.5 автоматически дополняется данными пулемёта. Купленное оружие, броня, уровни автомата, настройки чувствительности и FPS сохраняются.
