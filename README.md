# Boom Arena — Android FPS prototype 0.4

Самостоятельный мобильный FPS-прототип на **Godot 4.7.1**. Проект не использует код, модели, карты, названия оружия или другие ресурсы Guns of Boom.

## Что исправлено в 0.4

- полностью переделано сенсорное управление;
- добавлен плавающий левый джойстик с мёртвой зоной и плавным разгоном;
- обзор работает отдельным пальцем на правой части экрана и не конфликтует с кнопками;
- добавлена настройка чувствительности камеры в главном меню;
- камера получила более широкий FOV, плавную обработку свайпа и лёгкое покачивание при ходьбе;
- игрок при старте и после возрождения смотрит в сторону центра арены, а не в стену;
- исправлено направление движения относительно камеры;
- кнопка **ОГОНЬ** переведена на отдельный мультисенсорный обработчик;
- выстрел теперь заметен: вспышка у ствола, отдача, трассер, уменьшение патронов и маркер попадания;
- автострельба получила небольшую зону захвата цели около прицела;
- трассеры добавлены и ботам;
- интерфейс привязан к краям экрана и лучше работает на широких телефонах;
- включён поворот между обеими альбомными ориентациями телефона;
- в репозиторий добавлен постоянный прототипный ключ подписи, чтобы следующие APK обновлялись без удаления игры.

## Уже реализовано

- матч **4 на 4**: игрок + 3 союзных бота против 4 ботов;
- автомат **AR-4 Штурм**, дробовик **SG-12 Гром**, резервный пистолет и нож;
- четыре оружейных слота;
- автострельба с возможностью отключения;
- монеты, покупка дробовика и улучшение оружия;
- сохранение прогресса и чувствительности;
- ограничение до 60 FPS;
- автоматическая сборка и проверка ARM64 APK через GitHub Actions.

## Управление Android

- движение — нажать и вести пальцем в левой нижней части экрана;
- камера — свайпать свободным пальцем по правой части экрана;
- ручная стрельба — удерживать кнопку **ОГОНЬ**;
- автоматическая стрельба — кнопка **AUTO ON/OFF**;
- отдельные кнопки перезарядки, ножа и оружейных слотов 1–4.

## Загрузка и сборка

1. Скопируй всё содержимое проекта в свой репозиторий, включая скрытую папку `.github` и папку `signing`.
2. Запусти `ОБНОВИТЬ_GITHUB_0.4.bat` либо выполни `git add -A`, `git commit`, `git push`.
3. Открой **GitHub → Actions → Build Android APK**.
4. После зелёной сборки скачай artifact `BoomArena-Android-APK`.

### Важное про установку 0.4

Версии 0.1–0.3 собирались временным ключом GitHub Actions, который не сохранялся. Поэтому перед первой установкой 0.4 старую Boom Arena потребуется удалить один последний раз. Начиная с 0.4 используется постоянный ключ из `signing/boomarena-debug.keystore`, и последующие сборки смогут устанавливаться обычной кнопкой **«Обновить»**.

Ключ находится в репозитории только для тестового прототипа. Для публикации игры в магазине потребуется отдельный секретный release-ключ.
