# Обновление Boom Arena до 0.8.2

1. Распакуйте архив.
2. Скопируйте всё содержимое папки `BoomArena_GitHub_0.8.2` в локальный репозиторий с заменой.
3. Убедитесь, что перенеслась скрытая папка `.github`.
4. Запустите `ОБНОВИТЬ_GITHUB_0.8.2.bat`.
5. В GitHub Actions должны последовательно пройти шаги:
   - `Import Godot resources`;
   - `Run real battle scene smoke test`;
   - `Export Android APK`.
6. В `smoke-test.log` должен присутствовать маркер `BOOM_ARENA_SMOKE_TEST_OK`.

Версия 0.8.2 подписывается тем же ключом и использует тот же package ID, поэтому устанавливается поверх 0.8/0.8.1 как обновление, если предыдущая APK была собрана из этого репозитория с тем же ключом.
