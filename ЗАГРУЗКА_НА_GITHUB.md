# Первая загрузка проекта на GitHub

1. Создайте на GitHub пустой репозиторий, например `BoomBoom`.
2. Распакуйте проект в отдельную папку.
3. Запустите `ЗАГРУЗИТЬ_НА_GITHUB.bat`.
4. Вставьте HTTPS-адрес пустого репозитория.
5. После отправки откройте вкладку **Actions**.
6. Workflow соберёт подписанный APK и поместит его в артефакт `BoomArena-Android-APK`.

Не удаляйте скрытую папку `.github`, папку `signing` и файл `export_presets.cfg`.
