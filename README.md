# Boom Arena — Android FPS prototype

Самостоятельный мобильный FPS-прототип на **Godot 4.7.1**. Проект не содержит ресурсов, названий, карт или кода Guns of Boom.

## Что реализовано в 0.1

- матч **4 на 4**: игрок + 3 союзных бота против 4 ботов;
- вид от первого лица и простая 3D-арена;
- управление Android: виртуальный стик, свайп для обзора, огонь, перезарядка, нож, 4 слота;
- управление ПК для проверки: `WASD`, мышь, ЛКМ, `R`, `Q`, `1–4`, `F`;
- автострельба по противнику под прицелом, которую можно выключить;
- два основных оружия: автомат **AR-4 Штурм** и дробовик **SG-12 Гром**;
- резервный пистолет при полном окончании патронов основного оружия;
- отдельный удар ножом;
- монеты за убийства и завершение матча;
- покупка дробовика, улучшение урона оружия до 5-го уровня;
- четыре настраиваемых оружейных слота, которые можно оставлять пустыми;
- сохранение монет, покупок, улучшений, снаряжения и автострельбы;
- ограничение 60 FPS;
- GitHub Actions собирает ARM64 APK.

## Сборка через GitHub

1. Создайте пустой репозиторий.
2. На Windows запустите `ЗАГРУЗИТЬ_НА_GITHUB.bat` либо загрузите **содержимое** архива вручную в корень репозитория.
3. Откройте вкладку **Actions**.
4. Выберите **Build Android APK** → **Run workflow**.
5. После сборки откройте запуск workflow и скачайте артефакт `BoomArena-Android-APK`.

Workflow использует Godot Android Export action. Для локального экспорта Godot рекомендует JDK 17 и установленный Android SDK. Официальная документация: раздел `Exporting for Android` на сайте Godot.

## Открытие проекта локально

- Установите Godot 4.7.1 Standard.
- Нажмите **Import** и выберите `project.godot`.
- Запустите проект кнопкой **F6/F5**.

## Важные ограничения прототипа

Это первая техническая версия. В ней пока нет:

- сетевого сервера и реальных игроков;
- анимаций рук и персонажей;
- качественных моделей оружия и карты;
- звуков, музыки, отдачи камеры и эффектов попаданий;
- полноценного поиска пути у ботов;
- античита, аккаунтов и серверной экономики.

Сетевую часть нужно добавлять после стабилизации локального боя и управления.
