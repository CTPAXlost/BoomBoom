# История изменений

## 0.9.0 — тактический темп, полигон и снаряжение

- В «Салуне» оставлена одна контрольная точка в центре здания.
- Исправлена тактика ботов: часть атакует точку, часть прикрывает входы; после захвата только один остаётся защитником.
- Скорость игрока и ботов снижена, ускорение и торможение сглажены.
- Главное меню разделено на вкладки: бой, полигон, магазин, настройки и админ.
- Добавлен полигон с пассивными ботами, выбором любого оружия и бесконечными расходниками.
- Добавлен редактор свободного расположения мобильных кнопок с сохранением и сбросом.
- Добавлена светошумовая граната с 3 уровня: белый экран до 3 секунд, проверка направления и укрытия, ослепление ботов.
- Добавлен ремкомплект с 3 уровня: цена 2000 монет, восстановление 100 брони.
- Улучшены мультяшные модели персонажей, анимация ходьбы и составные модели оружия.
- Улучшены освещение, сглаживание и оформление карт на высоком качестве.
- GitHub Actions запускает отдельные smoke tests обычного матча и полигона.
- Android: версия 0.9.0, versionCode 11.
